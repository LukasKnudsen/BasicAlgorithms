# MergeSort (Sorting)

This folder demonstrates **MergeSort**

## Concept

MergeSort works by:

1. Dividing the array into 2 halves
2. Recursively sorting each half
3. Merging the two sorted halves into one sorted array

## Complexity

- Best case: **O(n log n)**
- Average case: **O(n log n)**
- Worst case: **O(n log n)**
- Extra space: **O(n)**
