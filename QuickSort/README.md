# QuickSort (Sorting)

This folder demonstrates **QuickSort**, a comparison-based sorting algorithm.

## Concept
Works by:
1. Choosing a pivot
2. Partitioning the array into elements ≤ pivot and > pivot
3. Sorting the two partitions

## Complexity
- Best / Average: **O(n log n)**
- Worst case: **O(n²)** (can happen with bad pivot choices)
- Extra space: **O(log n)** expected (recursion stack)
